# ArchInstaller
A simple Arch Linux Installer

## Guide
* Install git

    `sudo pacman -Syy git`

* Clone the repo
    
    `git clone https://github.com/alemazzo/archinstaller`

* Change directory

    `cd ArchInstaller`

* Make the script executable

    `chmod +x ./installer.py`

* Execute the script

    `sudo ./installer.py`
