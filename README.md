# ArchInstaller
A simple Arch Linux Installer

## Guide
* Install git

    `sudo pacman -Syu git`

* Clone the repo
    
    `git clone https://github.com/alemazzo/archinstaller`

* Change directory

    `cd archinstaller`

* Setup your configuration
  
    `nano config.yml`

* Make the script executable

    `chmod +x ./installer.py`

* Execute the script

    `sudo ./installer.py`
